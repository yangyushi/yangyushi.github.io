我在 Ubuntu 系统（20.04）下安装了 Steam， 并且使用其内置的 proton 程序运行了游戏「街头霸王 5 」（SFV）。 可惜我的系统语言是英文，这导致游戏的内置语言也是英文。 这个博客介绍了将游戏语言设置成中文的方法。

游戏文件的位置

我的电脑硬件包含有「内置固态硬盘」以及「额外的机械硬盘」。 我的游戏都安装在后者。在 Ubuntu 中，我的 Steam 游戏的路径是

/media/yushi/hdd/Steam

我的 SFV 有一个表示它的 ID：310950，这一串数字很重要。

通常的修改语言方法

在网络上，常见的修改语言文件的方法如下。 （参考网页） 可惜这个方法只适用于 Windows 系统。

  1. 找到文件夹 %appdata% > Local > StreetFighter > Saved > Config > Windowsnoeditor
  2. 修改该文件夹下的文件 Engine.ini，并且加入下面的内容
[Internationalization]
Culture=zh-cn

这个方法的逻辑是正确的： 为了修改 SFV 的语言，我们需要 hack Engine.ini 这个文件，额外加入「国际化–中文」的设置。

在 Ubuntu 中修改语言

在 Ubuntu 里，我们需要找到 Engine.ini 这个文件。 在我的系统下，它在下面的（很长的）路径里。为了找到它，我参考了这个帖子。我相信，你也可以使用 Unix 的 find 命令找到这个文件夹。

/media/yushi/hdd/Steam/steamapps/compatdata/310950/\
pfx/derive_c/users/steamuser/Local Settings/\
Application Data/StreetFighterV/Saved/Config/WindowsNoEditor

这个路径可以分解为下面几部分

/media/yushi/hdd/Steam/ # Steam 游戏的位置，我的机械硬盘
steamapps/compatdata/310950/  # SFV 的游戏数据位置，310950 是 SFV 的 ID
derive_c/users/steamuser/Local Settings/Application Data/  # 等价于 Windows 里的 %appdata%
StreetFighterV/Saved/Config/WindowsNoEditor  # engine.ini 的位置

同样地，我修改该文件夹下的文件 Engine.ini，并且加入下面的内容：

[Internationalization]
Culture=zh-cn

再启动 SFV，我的界面语言变成中文了！

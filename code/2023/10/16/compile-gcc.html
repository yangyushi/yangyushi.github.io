Sometimes we need our own version of gcc (often the latest versions) to use the new features in C++. This means we need to compile the souce code on the machine, which can be difficult.

General Steps

  1. Download the source code.
  2. Perform a out-of-source build.
  3. Installing dependencies:
    • libmpc
    • mpfr
    • gmp
    • isl
  4. Configure.
  5. Compile (make).

Notes

  • We need to unset C/C++ related flags with command unset.
